Installation

Unlicode is available for Windows, macOS, and Linux. Choose your platform below:

macOS

1

Download the latest .dmg file

2

Drag Unlicode to Applications folder

3

Launch from Applications

Linux

1

Download the .AppImage file

2

Make it executable: chmod +x Unlicode.AppImage

3

Run: ./Unlicode.AppImage

Windows

1

Download the .exe installer

2

Run the installer as administrator

3

Follow the installation wizard

Quick Start

Get up and running with Unlicode in minutes:

1. Install Unlicode

Download and install Unlicode for your operating system using the instructions above.

2. Set Up Local LLM

Choose and download a local language model. We recommend starting with Qwen Coder or AIR M1.

3. Start Coding

Open a project, write some code, and use Cmd/Ctrl + K to activate the AI assistant.

Basic Usage Example

Python 
# Open Unlicode and create a new Python file
# Type your question or request in the AI panel
# Example: "Write a function to calculate fibonacci numbers"

def fibonacci(n):
    """Calculate the nth Fibonacci number."""
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# Use Cmd/Ctrl + K to ask the AI to:
# - Optimize this function
# - Add error handling
# - Write tests
# - Explain the algorithm

Local LLMs

Unlicode supports a wide range of local language models. Here are our recommended options:

Qwen Coder

Excellent for code generation and understanding. Good performance on CPU.

  • Size: 7B parameters
  • RAM: 8GB minimum
  • Performance: Fast inference
Download Model

AIR M1

Optimized for Apple Silicon. Excellent performance on M1/M2 Macs.

  • Size: 3B parameters
  • RAM: 4GB minimum
  • Performance: Native M1 optimization
Download Model

CodeLlama

Meta's code-focused model. Great for complex programming tasks.

  • Size: 7B parameters
  • RAM: 16GB minimum
  • Performance: High quality output
Download Model

Code Generation

Generate complete functions, classes, and modules based on your specifications.

Code Review

Debugging

Identify and fix bugs with intelligent error analysis and suggestions.

Privacy & Security

Your privacy is our top priority. Here's how Unlicode protects your data:

Local Processing

All AI processing happens on your machine. Your code never leaves your computer.

No Data Collection

We don't collect, store, or transmit any of your code, prompts, or personal data.

Complete Anonymity

Use Unlicode without creating accounts or providing personal information.

Offline Capable

Once you have local models, Unlicode works completely offline.

Security Best Practices

  • Keep your local LLM models updated
  • Use trusted model sources
  • Regularly update Unlicode
  • Monitor system resources

Performance Tuning

Optimize Unlicode for your hardware setup:

CPU Optimization

  • Use smaller models (3B-7B parameters)
  • Enable CPU threading optimization
  • Close unnecessary applications
  • Use SSD storage for models

GPU Acceleration

  • Install CUDA drivers (NVIDIA)
  • Use Metal Performance Shaders (Apple)
  • Enable GPU memory optimization
  • Use larger models for better quality

Memory Management

  • Close unused browser tabs
  • Monitor RAM usage
  • Use model quantization
  • Restart Unlicode if needed
